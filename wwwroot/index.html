Swagger UI provides interactive documentation for all available API endpoints. Use it to:

  • Understand authentication requirements

Direct API Access

You can access the API endpoints directly. Here are some examples:

Get All Events:

GET http://localhost:5197/api/events

Get Event by ID:

GET http://localhost:5197/api/events/{id}

Create New Event:

POST http://localhost:5197/api/events

To test these endpoints, you can use tools like:

  • Postman
  • cURL
  • Your browser's developer tools
